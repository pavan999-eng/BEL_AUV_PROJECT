# BEL_AUV_PROJECT

## Overview
This project is an AI-based underwater object detection and classification system for AUV (Autonomous Underwater Vehicle) applications.

The system uses:
- YOLOv8 for object detection
- CNN/SVM models for classification
- Python and OpenCV for image processing

---

## Features
- Underwater object detection
- Real-time prediction
- CNN-based classification
- YOLO training and inference
- Image preprocessing

---

## Tech Stack
- Python
- OpenCV
- YOLOv8
- PyTorch
- scikit-learn
- NumPy

## Installation

Clone the repository:

```bash
git clone https://github.com/pavan999-eng/BEL_AUV_PROJECT.git
```

Move into the project folder:

```bash
cd BEL_AUV_PROJECT
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Usage

Run detection:

```bash
python detect.py
```

Run training:

```bash
python train.py
```
